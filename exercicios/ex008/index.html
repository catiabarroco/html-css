💪 Principais formatações 💪


💥 Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> e </b>
Esta tag não tem semantica, nao tem sentido

Nesta frase temos um termo em destaque usando a tag <strong> e </strong>
Esta forma tem semantica, é CORRECTA


💥 Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> e </i >
Esta tag não é semantica, não tem sentido

Nesta frase temos um termo com enfase usando a tag <em> e </em >
Esta forma tem semantica, é CORRECTA


💥 Texto Marcado

Podemos criar tambem um texto marcado usando a tag <mark> e </mark>
Podemos mudar a cor do texto usando css: Texto marcado <mark style="background-color: ...">e </mark>
Podemos tambem definir a cor de todos os mark's na parte do style do head


💥 Texto grande e Pequeno

Estamos criando um texto grande e um texto pequeno neste paragrafo
O texto big está obsoleto
<big> e </big> ; <small> e </small>


💥 Texto deletado

Podemos marcar um texto como excluido, para indicar que ele deve ser lido mas nao considerado com a tag <del> e </del>


💥 Texto inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que foi adicionado depois com a tag <ins> e </ins>.

Existe tambem o sublinhado com a tag <u> e </u> mas está obsoleta


💥 Texto Sobre-escrito

Para inserir coisas do tipo x20+3
Com as tags <sup> e </sup>


💥 Texto Sub-escrito

Para inserir coisas do tipo H20
Com as tags </sub> e </sub>
